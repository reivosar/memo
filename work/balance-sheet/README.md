| ---- | ---- |
|  総資産  |  総資産とは、会社が運用している財産の総額すなわち資産の合計を意味するものです。資産には「流動資産」や「固定資産」「繰延資産」などがありますが、これら会社のすべての資産を合算したものをいいます  |
